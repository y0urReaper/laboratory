Интерактивная лабораторная работа по физике

"Определение ускорения свободного падения на телах Солнечной системы"

Описание работы:
На данной странице представлена виртуальная лабораторная работа, которая наглядно иллюстрирует процесс определения ускорения свободного падения по заданным параметрам, в том числе, выбранным телам Солнечной системы. Может использоваться в качестве практической работы, для проведения расчётов, при выполнении лабораторной работы.

Цель работы:
научиться определять ускорение свободного падения на телах Солнечной системы, используя для этого лабораторную работу, а также формулы, изученные ранее.

Для более точного выполнения работы требуется обозначить максимальную высоту за 1,5 м. Шар, являющийся моделью тел Солнечной системы, располагается именно на этой высоте до выполнения работы.
